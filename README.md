## 🧩 Como Executar <a id="como-executar"></a>

Para executar o projeto na sua máquina, basta seguir a sequencia de passos:

```
# 1 - Clone o repositorio
# 2 - Baixe as dependencias
# 3 - [backend]$ yarn dev
# 4 - [frontend]$ yarn dev

* O banco de dados está hospedado externamente
```
